DevOps Tooling & Setup Guide
This repository contains the essential setup steps for my development environment, covering SSH, Markdown, and VS Code.

🛠 Tools Covered
SSH: Secure communication for remote access.

VS Code: The primary code editor for development.

Markdown: For clean, professional documentation.

🚀 Quick Install Commands
SSH (Linux/Ubuntu)
Bash

sudo apt-get install openssh-client openssh-server
SSH Key Generation
Bash

ssh-keygen -t rsa -b 2048 -C "your_email@example.com"
📝 Markdown Basics
Headings: Use # for big titles and ## for smaller ones.

Lists: Use a dash - at the start of a line.

Code: Wrap code in backticks like this: `code`.
